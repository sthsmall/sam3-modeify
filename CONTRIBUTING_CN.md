# 为 sam3 做贡献
我们希望使对这个项目的贡献变得简单透明。

## 拉取请求
我们积极欢迎您的拉取请求。

1. 分叉存储库并从 `main` 创建您的分支。
2. 如果您添加了应该测试的代码，请添加测试。
3. 如果您更改了 API，请更新文档。
4. 确保您的代码符合代码风格要求。
5. 如果您还没有完成贡献者许可协议（"CLA"）。

## 贡献者许可协议（"CLA"）
为了接受您的拉取请求，我们需要您提交 CLA。您只需要这样做一次，就可以在 Facebook 的任何开源项目上工作。

在这里完成您的 CLA：<https://code.facebook.com/cla>

## 问题
我们使用 GitHub 问题来跟踪公共错误。请确保您的描述清晰，并提供足够的说明来重现问题。

Facebook 有一个 [奖励计划](https://www.facebook.com/whitehat/) 用于安全披露安全漏洞。在这些情况下，请按照该页面上概述的流程进行操作，不要提交公共问题。

## 许可证
通过为 sam3 做出贡献，您同意您的贡献将在该源代码树根目录中的 LICENSE 文件下获得许可。
